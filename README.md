# Industrial PLC & Automation Portfolio

این مخزن به صورت تخصصی بر برنامه‌نویسی کنترل‌کننده‌های منطقی برنامه‌پذیر (PLC)، طراحی سیستم‌های اتوماسیون صنعتی و نقشه‌کشی تابلوهای برق متمرکز است.

## حوزه‌های تخصصی:
- **برنامه‌نویسی PLC:** توسعه کدهای بهینه برای پروسه‌های صنعتی (بر پایه استاندارد IEC 61131-3).
- **طراحی تابلو برق:** نقشه‌کشی مدارات قدرت و فرمان، انتخاب تجهیزات و مدیریت باس‌بار.
- **HMI & SCADA:** طراحی رابط کاربری برای مانیتورینگ سیستم‌های صنعتی.
- **اتوماسیون:** پیاده‌سازی سیستم‌های کنترل فیدبک و مدیریت خطوط تولید.

## ساختار فایل‌ها:
- `/docs`: مستندات فنی، کاتالوگ‌ها و گزارش‌های پروژه.
- `/diagrams`: نقشه‌های فنی (EPLAN یا AutoCAD) و دیاگرام‌های سیم‌کشی.
- `/src`: سورس‌کدهای برنامه‌نویسی (TIA Portal, Step7, Codesys و غیره).
- `/images`: مستندات تصویری از پروژه‌های اجرا شده.

## نحوه استفاده از کدها:
تمامی کدهای موجود در این مخزن برای اهداف آموزشی و صنعتی تهیه شده‌اند. لطفاً قبل از بارگذاری کد روی سخت‌افزار واقعی، حتماً از شبیه‌سازها (PLCSIM) استفاده کنید.
